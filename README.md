AI Chat Application 🤖💬

I’m proud to share my first full-stack AI chat application!  
Built from scratch with no prior full-stack experience, this project taught me the fundamentals of connecting front-end and back-end systems and integrating real-world APIs.

---

## Features ✨
- **Custom Front-end UI** – HTML, CSS, JavaScript  
- **Backend** – Node.js + Express  
- **API Integration** – OpenAI  
- **Async/Await + Fetch requests** for smooth communication  
- **Environment Variables** – secure API handling  
- **Real-time message rendering** – dynamic DOM updates without page refresh  
- Debugged and deployed locally

## How to Run 🏃‍♂️

Follow these steps to run the AI Chat website on your computer:

1. **Get the project files from GitHub**  
   Open your terminal or command prompt and type the following command:
   
   git clone https://github.com/ar-eej/ai-chat-app.git

   This will download the project to a folder called `ai-chat-app`.

2. **To Go into the project folder (eg: VScode)**
   Type the following command in your terminal:
   
   cd ai-chat-app

4. **Install the necessary dependencies**
   Type the following command in your terminal:

   npm install

   This installs all the tools your project needs to run.

5. **Add your OpenAI API key**  
   Create a file named `.env` in the project folder and add this line:

   OPENAI_API_KEY=your_api_key_here
   Replace `your_api_key_here` with your actual OpenAI API key.
  
5. **Start the server**
   Type the following command in your terminal:
   
   node server.js

   This runs the backend so your chatbot works locally.

6. **Open the website in your browser**
   
   Go to: [http://localhost:4000](http://localhost:4000)  
  
   You should see your AI chat application ready to use!
